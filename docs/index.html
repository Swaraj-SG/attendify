Features

😁

Real Prediction

Predicts future potential lectures which can be bunked. (The number of bunkable lectures may vary in actual number)

⚡

Fast Calculations

Blazing fast calculation speed, and auto saves the entered attendance data.

🛡️

Secure Storage

Data is stored locally in .json format. Local storage is ideal for people who value privacy.

How it works

  1. It gets installed and creates a .json file 📥
  2. Saves the attendance data in readable format 🧾
  3. Anyone can reset the attendance data by deleting the .json file. The app will create it again. 🗂️
  4. The app uses a speacial formula to calculate the attendance and displays it along with future possible bunks. ⚙️
  5. Shows the total % of attendance with color warnings in real time. 🔄
  6. Calculates the attendance percentages on 75% as target due to most colleges/universities have 75% attendance requirement. 📊

About Attendance Bunker

Attendance Bunker is a focused tool to help students manage their attendance accurately and privately.
Built with simplicity and speed in mind.

Key properties

  • Calculates attendance fast and accurately.
  • Auto-saves the entered attendance data.
  • Offline functionality — stores the data locally on device.
  • Simple and user-friendly UI.

Future ideas

  • Realtime sync across multiple devices.
  • To shrink the app size and overhaul the UI.
  • To create import/export feature for attandance data.
